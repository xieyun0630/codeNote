### windows设置开机自启动

设置启动项

1.找到启动文件夹，我的是C:\Users\ThinkPad\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup，或者打开运行，输入shell:startup，回车，也可以快速打开启动文件夹。
